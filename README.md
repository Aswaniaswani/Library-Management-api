 Library Management System API
 
A Library Management System REST API built using Python and Django (Django REST Framework). This API allows managing library resources such as books, users, and transactions programmatically, making it suitable for integration with web, mobile, or third-party applications.

Features

* Authentication & Authorization (Admin / User)
* Create, Read, Update, Delete (CRUD) Books
* Search & Filter Books
* Issue Books to Users
* Return Books
* Track Book Availability
* Maintain Issue & Return Records
* RESTful API design

Tech Stack

Backend: Python, Django
API Framework: Django REST Framework (DRF)
Database: SQLite (default)
API Format: JSON
